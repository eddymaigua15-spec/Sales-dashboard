# Sales Dashboard App

## Overview
This is a Streamlit-based sales performance dashboard designed for tracking product, region, and time-based sales metrics.

## Features
- Total sales KPIs
- Units sold tracking
- Product performance analysis
- Regional performance breakdown
- Sales trend visualization

## Files
- app.py: Main Streamlit application
- sales_data.csv: Sample dataset
- requirements.txt: Dependencies

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run the app
streamlit run app.py

## Expected Output
- Interactive dashboard in browser at http://localhost:8501

## Business Use Case
Designed for agribusiness sales intelligence, territory management, and executive reporting.
